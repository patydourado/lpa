Este repositório contém o material didático, exemplos de código e exercícios da disciplina de **[Lógica de Programação e Algoritmos(LPA)]**, lecionada nos cursos de Computação.

---

## 🎯 Objetivos da Disciplina
Preparar o aluno para a elaboração de algoritmos voltado para solução de problemas tratados a partir da interpretação da lógica computacional. Além disso, busca-se introduzir um ambiente de programação 
para uso e implementação de soluções com a linguagem Java

## 📂 Organização do Repositório (Em construção...)
Para manter a organização, os arquivos estão divididos da seguinte forma:

* `/exemplos`: Códigos-fonte demonstrados em sala de aula.
* `/exercicios`: Listas de exercícios e enunciados de trabalhos.
* `/recursos`: Imagens e diagramas utilizados na documentação.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** [Portugol, Java]
* **IDE Sugerida:** [Portugol Studio, Eclipse]
